# 🏡 Homestay Impian Enterprise

Sistem Laman Web Tempahan Homestay Mudah bersepadu dengan **Google AdSense** dan sistem pesanan **WhatsApp API**. Laman web ini dibina khusus bagi memudahkan pelanggan menempah pakej penginapan di Kota Marudu, Sabah.

## 🚀 Ciri-Ciri Utama
* **Integrasi WhatsApp:** Borang tempahan automatik menukar input pelanggan ke dalam bentuk teks kemas dan dihantar terus ke WhatsApp admin (`60177241468`).
2* **Sedia Google AdSense:** Ruang iklan khusus yang responsif untuk menjana pendapatan pasif (Mobile-friendly).
* **Had Tarikh Pintar:** Skrip JavaScript secara automatik menghalang pelanggan daripada memilih tarikh *check-in* yang sudah lepas (tarikh semalam/lalu).
* **Senarai Pilihan Fleksibel:** Menawarkan 3 jenis penginapan utama (Goshen, Villa Pool, & Condo Premium).

## 📍 Lokasi & Kemudahan
* **Alamat:** Kampung Goshen jalan Lotong, 89100 Kota Marudu, Sabah, Malaysia.
* **Kemudahan standard:** Bilik tidur berhawa dingin (Aircond), TV, Wi-Fi percuma, dapur memasak, peti sejuk, dan ruang parkir yang luas.

## 🛠️ Teknologi Yang Digunakan
* **HTML5:** Struktur kandungan dan borang kemasukan data.
* **CSS3:** Reka bentuk grid responsif, kad produk moden, dan reka letak mesra peranti mudah alih.
* **JavaScript (Vanilla):** Logik pengiraan tarikh minimum dinamik serta format penyusunan mesej WhatsApp API.
